CEP-DIP Fixed-Density Solvent-Field Reproduction — cal_18 Validation

Starting from a converged CHGCAR, the implicit-solvent fields (PHI / RHOB / RHOION) are recomputed twice — with a patched VASP/CEP-DIP build and with a pure-Python nonlinear Poisson–Boltzmann solver — and compared term by term against the original DFT reference. This page collects every numerical conclusion kept under results/ and comparison/ for review before anything goes into the repository.

case: cal_18 (Ni₁N₁-doped graphene slab) grid: 168 × 168 × 384 c axis: 35 Å VASPsol: ISOL=2, TAU=9e-3, LDIPOL(z)

Three-Stage Verification Chain

Each stage has its own numerical criterion; every stage builds on the one before it.

STEP 1

VASP fixed-CHGCAR reproduction

Patched CEP-DIP skips the charge position correction and recomputes the solvent fields from the original inputs plus the final CHGCAR.

CHGCAR byte-identical · PHI RMSE 1.7e-7 eV
STEP 2

Python input terms aligned

Valence charge, DENCOR core correction, CVHAR, and the Ewald/CDIPOL dipole terms are each matched to patched-VASP debug output.

NVAL 5e-13 · CVDIP 2e-12 eV · EF_z 1e-13
STEP 3

Python end-to-end Newton solve

RHOB / RHOION / PHI computed directly from CHGCAR/POTCAR/config and compared against the reference fields.

PHI 3D RMSE 3.0e-3 eV · RHOB 1D 3e-6 e/Å³

Confirmed negative result: the earlier "planar-average first, then solve a 1D nonlinear PB" shortcut is not viable — planar averaging does not commute with the nonlinear dielectric response, and the bound charge can be off by orders of magnitude. Any future dimensional reduction should start from the 3D baseline on this page and be compared numerically against the reference fields, term by term.
